                                      • 变分自编码器 VAE

                                      直观理解

                                      从图像开始

                                      变分自编码器（Variational Auto-Encoder，VAE），是现代 Stable Diffusion 架构的关键组成部分。而所谓自编码器（Auto-Encoder）即接收数据，将其压缩为低维表示，然后将其重构回原始形式。以图片为例，虽然图像包含数百万个值，但其低维表示往往是一个几十或者上百维的向量，这些表示存在于一个更加紧凑（compact）空间中，通常称之为潜在空间（Latent Space）。

                                      现在假设我们已经有了一个训练好的自编码器，能够从潜在表示（Latent Representation）中重构图像，那么生成新的数据的一个自然的想法就是从潜在空间中随机采样，然后通过解码器（Decoder）运行他们。但是问题是，一般意义下的潜在空间的结构是很糟糕的，这意味着即使采样的两个点相近，它们所获得的编码图像也可能完全不同，可以说潜在空间是无组织且不规则的。

                                      理想情况下，我们希望存在一个良好组织的潜在空间，其中的采样点可以产生连贯的新图像。如何去实现呢？

                                      让我们来看看 VAE 是如何做的。

                                      贝叶斯观点

                                      我们的目标是从给定的分布 p(x) 中生成新的数据，它代表我们的数据集，比如上面提到的图像。 问题在于我们并不清楚 p(x) 的具体形状或者性质，而是只能够获取一些样本，比如来自训练集的图像。

                                      为了使得处理 p(x) 更加容易，我们引入另外一个分布 p(z) ，称为潜在分布（latent distribution），该分布表示潜在变量，表示低维空间中的向量。由于这两个分布在不同的空间当中，因此我们需要映射来连接它们。这里我们有两个映射：

                                      1. 第一个映射称为后验分布 p(z|x) ，它表示给出特定图像 x 后潜在向量 z 生成的概率。

                                      2. 第二个映射是似然分布 p(x|z) ，它表示给定潜在向量 z ，从中重建图像 x 的概率。

                                      由于不清楚潜在分布 p(z) 的确切形式，我们假设它为一个标准正态分布 N(0,1) ，这个假设允许我们去计算似然 p(x|z) 。而对于后验分布，由于不清楚真实的后验，我们使用高斯分布来近似它，称其为 q(z|x)=N(μ,σ2) ，其中的 μ 和 σ 都是我们需要学习的参数，这是一个优化过程，通常称为变分贝叶斯。我们将训练一个深度编码器来从图像当中估计这些参数，然后，我们使用解码器从近似后验（高斯分布）中采样的潜在变量重构图像。

                                      那么我们如何实际训练这个自编码器来近似后验分布，然后从中重构图像呢？

                                      损失函数

                                      我们的目标是最大化 log⁡p(x) ，这可以写为

                                      log⁡p(x)=log⁡∫p(x,z)dz=log⁡∫q(z|x)q(z|x)p(x,z)dz=log⁡Eq(z|x)[p(x,z)q(z|x)]≥Eq(z|x)[log⁡p(x,z)q(z|x)],

                                      上面的不等式利用了上凸函数的 Jensen 不等式 E[f(x)]≤f(E[x]) 。

                                      不等式的右侧被称为证据下界（evidence lower bound, ELBO），如果我们希望最大化对数似然 log⁡p(x) ，那么我们可以最大化 ELBO 。利用 p(x,z)=p(x|z)p(z) ，ELBO 可以被分解为两个主要部分：

                                      (1)L(x)=Eq(z|x)[log⁡p(x|z)+log⁡p(z)q(z|x)].

                                      第一部分是似然函数在后验分布上的期望，而第二项实际上就是后验分布 q(z|x) 和先验分布 p(z) 之间的 KL 散度，即损失函数可以写为

                                      (2)L(x)=Eq(z|x)[log⁡p(x|z)]数据一致性−KL(q(z|x)|p(z))正则化.

                                      Note
                                      实际上第一项在特定的假设下可以表示为均方误差（MSE），假定解码器输出的分布为 p(x|z)=N(x;μθ(z),σ2) ，即方差是固定的，那么

                                      (3)log⁡p(x|z)=−12σ2∥x−μθ(z)∥2−D2log⁡(2πσ2)，

                                      其中 D 为解码器输出的均值向量的维度，μθ(z) 是解码器输出的均值。最大化第一项 Eq(z|x)[log⁡p(x|z)] 就等价于最小化重构误差 ∥x−μθ(z)∥2 。

                                      Note
                                      对于第二项的 KL 散度，它衡量了两个分布之间的近似程度，这意味着我们选择 p(z) 为正态分布的时候，当优化 ELBO 时，我们需要约束近似后验也采用正态分布的形状，这恰好就是我们前面说的。

                                      现在我们有了一个需要优化的目标，那么自编码器在实践中如何实现这些分布呢？

                                      回顾自编码器的作用，它接收输入，将其压缩为低维表示，然后将输入解码回输入空间。在标准的自编码器当中，这种压缩表示是潜在空间中的单个点，但是对于 VAE ，情况有一些不同。VAE 中编码器不是将输入映射到单个点，而是将输入转化为概率分布（例如高斯分布）。此时我们的图像不是潜在空间中的某个点，而是一个高斯分布。在这个潜在分布中，我们随机采样点，解码器将这些采样点转换回输入空间。

                                      然而，一个重要的问题是我们这里的采样操作无法进行反向传播 ，因为采样是不可微的。此时我们需要利用到重参数化技巧（reparameterization），我们不是直接从高斯分布采样，而是引入一个随机变量 ε ，在网络外部处理随机性。

                                      分布标准化

                                      前面我们进行的推导都是基于一个假设：p(z) 为一个标准正态分布。

                                      让我们思考一下为什么要这样做？对于一个生成模型的训练而言，我们的目标当然是样本 X 的重构值 X^ 与原样本的差别充分的小。然而这个过程受到噪声的影响，这是因为我们的潜在变量 Z 不是通过 encoder 算出来的，而是重新采样出来的，那么采样中的噪声就会增加重构的难度，但是另一方面，这个噪声的强度（方差）是通过一个网络计算出来的，为了最终模型重构的好，需要想办法让方差为 0 。但是方差为 0 的话，生成模型就没有随机性了，那么便不能称之为一个生成模型。

                                      VAE 实际上希望所有的后验分布 p(z|x) 都向标准正态看齐，这样就防止了噪声为零，同时保证了 p(z) 也为一个标准正态分布：

                                      (4)p(z)=∑xp(z|x)p(x)=∑xN(0,1)p(x)=N(0,1)∑xp(x)=N(0,1).

                                      那么如何让所有的后验分布 p(z|x) 都向标准正态看齐呢？最直接的方式应该是在损失函数中提到的 KL 散度，并且我们还有

                                      KL(N(μ,σ2)∥N(0,1))=∫12πσ2e−(x−μ)2/2σ2log⁡(e−(x−μ)2/2σ2/2πσ2e−x2/2/2π)dx=∫12πσ2e−(x−μ)2/2σ2log⁡{1σ2exp⁡{12[x2−(x−μ)2/σ2]}}dx=12∫12πσ2e−(x−μ)2/2σ2[−log⁡σ2+x2−(x−μ)2/σ2]dx,

                                      上式第一项为 −log⁡σ2 乘以概率密度的积分，也就是 −log⁡σ2 ；第二项为正态分布的二阶矩，为 μ2+σ2 ；第三项为 “负的方差除以方差” 为 -1 ，所以总的结果就是

                                      (5)KL(N(μ,σ2)∥N(0,1))=12(−log⁡σ2+x2−(x−μ)2/σ2).

                                      从联合分布推导损失函数

                                      假设样本的总分布为 p(x) ，我们希望用借助潜在变量 z 去描述 p(x) ：

                                      (6)q(x)=∫q(x|z)q(z)dz,q(x,z)=q(x|z)q(z).

                                      这里 q(z) 为先验分布（标准正态分布），目的为希望 q(x) 逼近 p(x) 。

                                      定义 p(x,z)=p(x)p(z|x) ，我们设想用一个联合分布 q(x,z) 逼近 p(x,z) ，那么我们可以用 KL 散度：

                                      DKL(p(x,z)∥q(x,z))=∬p(x,z)log⁡p(x,z)q(x,z)dzdx=∫p(x)(∫p(z|x)ln⁡p(x)p(z|x)q(x,z)dz)dx=Ex∼p(x)[∫p(z|x)ln⁡p(x)p(z|x)q(x,z)dz],

                                      由于有

                                      (7)Ex∼p(x)[∫p(z|x)ln⁡p(x)dz]=Ex∼p(x)[ln⁡p(x)∫p(z|x)dz]=Ex∼p(x)[ln⁡p(x)],

                                      而 p(x) 是确定的，即使我们可以不知道它的具体形式，因此这一项为一个常数。从而损失函数可以写为

                                      (8)L=DKL(p(x,z)∥q(x,z))−常数=Ex∼p(x)[∫p(z|x)ln⁡p(z|x)q(x,z)dz].

                                      可以知道最小化 KL 散度就相当于最小化上面的损失函数，损失函数还可以进一步写为

                                      L=Ex∼p(x)[∫p(z|x)ln⁡p(z|x)q(x,z)dz]=Ex∼p(x)[∫p(z|x)ln⁡p(z|x)q(x|z)q(z)dz]=Ex∼p(x)[−∫p(z|x)q(x|z)dz+∫p(z|x)ln⁡p(z|x)q(z)dz]=Ex∼p(x){Ez∼p(z|x)[−ln⁡q(x|z)]+KL(p(z|x)∥q(z))},

                                      可以看出来上面花括号中间的项正是我们上面推导的 VAE 损失函数，只是符号有一些不同。

                                      实验

                                      构建一维混合高斯数据集

                                      我们使用如下的一维混合高斯分布作为真实数据分布：

                                      (9)x∼12N(−2,0.52)+12N(2,0.52).

                                      该分布具有以下特点：

                                      • 非单峰（multi-modal）

                                      • 非线性结构

                                      • 分布形式完全已知

                                      这使得它非常适合用来检验潜在变量是否真的学到了“生成结构”，随机采样是否能复现真实分布。实验中生成约 10,000 个样本，并绘制其直方图作为基准分布。该分布在后续所有实验中都将作为“真值”进行对照。

                                      Encoder

                                      对于每一个输入样本 x，我们使用一个神经网络来近似其后验分布：

                                      (10)qϕ(z|x)=N(μϕ(x),σϕ2(x)).

                                      这里我们可以看到，VAE 输出分布的参数，而不是一个确定的 z ，这是 VAE 与普通 Auto-Encoder 的根本区别。Encoder 的结构为两层全连接网络加上 ReLU，输出分布的均值和对数方差。

                                      class Encoder(nn.Module):
                                          def __init__(self, hidden_dim=32):  # 编码器：将输入编码为潜在空间的均值和方差
                                              super().__init__()
                                              self.net = nn.Sequential(  # 1 -> 32 -> 32
                                                  nn.Linear(1, hidden_dim),
                                                  nn.ReLU(),
                                                  nn.Linear(hidden_dim, hidden_dim),
                                                  nn.ReLU(),
                                              )
                                              self.mu = nn.Linear(hidden_dim, 1)
                                              self.logvar = nn.Linear(hidden_dim, 1)  # 这里使用对数方差而不是方差，因为对数方差更容易优化
                                      

                                          def forward(self, x):
                                              h = self.net(x)  # 将样本输入 encoder，得到隐藏层输出 h
                                              mu = self.mu(h)
                                              logvar = self.logvar(h)
                                              return mu, logvar

                                      Decoder

                                      我们假设

                                      (11)pθ(x|z)=N(μθ(z),σx2),

                                      其中 σx2 为固定常数，正如上面所说，在这种情况下，最大化对数似然等价于最小化均方误差 MSE。Decoder 的结构也是两层全连接网络加上 ReLU ，输出为重构样本 x^ 。

                                      class Decoder(nn.Module):
                                          def __init__(self, hidden_dim=32):
                                              super().__init__()
                                              self.net = nn.Sequential(
                                                  nn.Linear(1, hidden_dim),
                                                  nn.ReLU(),
                                                  nn.Linear(hidden_dim, hidden_dim),
                                                  nn.ReLU(),
                                                  nn.Linear(hidden_dim, 1)
                                              )
                                      

                                          def forward(self, z):
                                              return self.net(z)  # 将潜在变量输入 decoder，得到重构数据

                                      VAE

                                      一个总的 VAE 实际上就是上述的 Encoder + Decoder，注意在对潜在分布进行采样的时候我们还需要利用重参数化技巧。

                                      def reparameterize(mu, logvar):
                                          std = torch.exp(0.5 * logvar)
                                          eps = torch.randn_like(std)
                                          return mu + eps * std

                                      那么一个总的 VAE 就是：

                                      class VAE(nn.Module):
                                          def __init__(self, hidden_dim=32):
                                              super().__init__()
                                              self.encoder = Encoder(hidden_dim)
                                              self.decoder = Decoder(hidden_dim)
                                      

                                          def forward(self, x):
                                              mu, logvar = self.encoder(x)
                                              z = reparameterize(mu, logvar)
                                              x_hat = self.decoder(z)
                                              return x_hat, mu, logvar, z

                                      ELBO

                                      VAE 的训练目标是最大化 ELBO：

                                      (12)L(x)=Eq(z|x)[log⁡p(x|z)]−KL(q(z|x)|p(z))，

                                      这里我们将其分开看为重构损失 Eq(z|x)[log⁡p(x|z)] 和 KL 散度 KL(q(z|x)|p(z)) 。在高斯似然的假设下，

                                      (13)Lrecon=12σx2|x−x^|2,

                                      而对于一维高斯分布，其 KL 散度可以计算为：

                                      (14)KL(N(μ,σ2)|N(0,1))=12(μ2+σ2−log⁡σ2−1).

                                      ELBO 的代码实现如下：

                                      def reconstruction_loss(x, x_hat, sigma_x=0.1):
                                          return ((x - x_hat) ** 2).mean() / (2 * sigma_x ** 2)  # 重构损失，在高斯似然的假设下为 MSE 损失
                                      

                                      def kl_divergence(mu, logvar):
                                          return 0.5 * (mu.pow(2) + logvar.exp() - logvar - 1).mean()  # KL 散度
                                      

                                      def vae_loss(x, x_hat, mu, logvar, beta=1.0):
                                          recon = reconstruction_loss(x, x_hat)
                                          kl = kl_divergence(mu, logvar)
                                          return recon + beta * kl, recon.detach(), kl.detach()  # detach 断开计算图，不需要计算梯度，只需要记录，节省内存

                                      Note

                                      默认设置 β 为1，此时为标准 VAE；若设置 β 为 0 ，则退化为普通的 AE。

                                      实验结果

                                      损失函数

                                      在训练过程中，重构损失逐渐下降，但不会趋近于 0 ；KL 损失收敛到一个稳定的正值。VAE 主动牺牲部分重构精度，换取潜在空间的可生成性。

                                      潜在变量分布

                                      潜在变量 z 的整体分布近似于 N(0,1) ，这说明 KL 散度生成约束了潜在空间。

                                      生成能力

                                      利用重参数技巧从 z∼N(0,1) 中独立采样，通过 Decoder 生成新样本，可以得到与真实数据分布高度一致的双峰结构，这验证了 VAE 的生成能力，即拟合原始数据分布。

                                      对照实验：β=0 （退化为 AE）

                                      当去除 KL 项的时候，即 β=0 ，实验结果如下。

                                      可以发现：

                                      1. 重构误差明显降低，并且趋近于 0。

                                      2. 潜在分布并不趋近于标准正态。

                                      3. 从 N(0,1) 采样无法生成合理的数据。

                                      这表明 KL 散度是 VAE 作为生成模型的结构性约束。

                                      基于 MNIST 数据集的实验

                                      MNIST 数据集

                                      MNIST 数据集包含60,000 张训练图像，10,000 张测试图像，每张图像大小为 28×2828×28，像素值范围为 [0,255] ，在本实验中，我们将其线性拉平成 784 维向量。因此与1D-VAE 相比，我们的数据维度急剧上升 x∈R→x∈R784 。

                                      实验结果

                                      • 当 β 很小（甚至 β=0）时，模型重构效果极好，生成图像清晰；

                                      • 当 β=1（标准 VAE）时，模型却只能生成几乎一模一样的、模糊的图像；

                                      • 无论如何进行 KL annealing、调网络结构，潜在变量 z 似乎总是被忽略。

                                      Posterior Collapse

                                      在实验中我们发现，Encoder 的输出总是一个与 x 无关的标准正态分布，即

                                      (15)μ(x)≈0,σ(x)≈1∀x,

                                      这意味着 q(z|x)≈p(z) ，也就是说 Decoder 的输出几乎与 z 无关，趋向于训练集的均值图像。这便是所谓的 posterior collapse ，潜在空间中无信息，尽管 z 是随机的，但是它与数据 x 已经完全无关。

                                      这实际上是 Loss Scaling 失衡导致的 Posterior Collapse，由于数据的维度是 784 维，当我们利用 mean 计算重构损失的时候，这会造成重构损失的量级远远小于 KL 散度的量级，此时重构误差计算的是每个像素的重构误差，而 KL Loss 是对 Latent Vector 的所有维度求和，因此我们只需要把被平均掉的像素数乘回来就行了！

                                      def reconstruction_loss(x, x_hat):
                                          return F.mse_loss(x_hat, x, reduction="mean")
                                      """改为"""
                                      def reconstruction_loss(x, x_hat):
                                          loss = F.mse_loss(x_hat, x, reduction='none') 
                                          loss = loss.view(loss.size(0), -1).sum(dim=1) # 对每张图的所有像素求和
                                          return loss.mean() # 对 batch 求平均
